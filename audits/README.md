# Quarterly Stability Audits

This directory contains quarterly stability audit reports generated by the automated audit workflow.

## Audit Schedule

Audits are run quarterly on the 15th of March, June, September, and December.

## Manual Audit Trigger

To manually trigger an audit:

```bash
# Via GitHub Actions UI
# Go to Actions > Quarterly Stability Audit > Run workflow
```

## Audit Reports

- Reports are named by quarter: Q1-2026.md, Q2-2026.md, etc.
- Each report includes metrics, security assessment, and quality analysis
- Reports are automatically committed and PR'd for review
